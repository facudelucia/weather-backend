# weather-backend

Demo of repo allocated in:
https://weather-facu.herokuapp.com

Current endpoints:
- /v1/location
- /v1/current/:city?
- /v1/forecast/:city?

To run local: 
- Clone repo
- Inside app folder run: 'npm install'
- To run in dev mode: 'npm run dev', app should be up in port 4000
- To run tests: 'npm test'
